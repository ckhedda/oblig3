Layouts i CSS

Relativ posisjonering

Et element med position: relativ er posisjonert relativt til den "normale" posisjonen. Vi posisjonerer disse i forhold til hvordan de ellers ville ha blitt plassert ellers med static.

Et element med position: fixed er låst. Det vil si at elementet står på den samme plassen uansett om siden scrolles ned eller ikke. top, right, bottom, og left er brukt for å posisjonere elementet.

Et element med position: absolute er posisjonert relativt til den nærmeste posisjonerte forgjengen (ancestor), for eksempel fixed.Absolutt posisjonering vil alltid posisjoneres ut fra elementet som ligger rundt, eller som nevnt foreldreelementet. For at dette skal fungere i praksis, er det viktig med en annen posisjonering enn static.

Elementer som overlapper hverandre

Elementer som overlapper med hverandre er et utfordring i seg selv. Derimot så går det an å løses ved å sette inn z-index på elementene.

CSS flexbox og CSS Grid

Istedenfor å gjøre posisjonering mer utfordrete og vanskeligere som det allerede er, kan man gjerne bruke CSS flexbox og CSS grid for å gjøre posisjoneringen enklere.

Som sagt kan CSS flexbox bli brukt til å posisjonere, men det kan også bli brukt til å lage bokser for nettsidens og designets skyld.

CSS grid er et "hjelpeverktøy" som skal hjelpe brukeren til å lage diverse design med rader og kolonner uten å bruke kommandoene float og position
